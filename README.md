# nikbucher's Homebrew Tap

Homebrew tap for my CLI tools.

## Install

```bash
brew tap nikbucher/tap
```

## Install Packages

```bash
brew install nikbucher/tap/<name>
```

## Packages

### Formulae

## Update / Uninstall

```bash
brew update
brew upgrade

brew uninstall <formula>
```
